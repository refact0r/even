# Even Realities G2 Smart Glasses Documentation

## Overview

The Even Realities G2 represents a privacy-first wearable platform featuring "dual micro-LED displays (one per lens), a four-microphone array, touchpads on the temples, and optional R1 ring for additional input."

## Hardware Specifications

| Component | Details |
|-----------|---------|
| Display | 576 x 288 pixels per eye |
| Color | 4-bit greyscale (16 shades of green) |
| Connectivity | Bluetooth 5.2 |
| Audio Input | 4-mic array (single stream, 16kHz PCM) |
| Touch Controls | Press, double press, swipe up/down on temples and optional ring |
| Notable Omissions | No camera or speaker |

## Development Capabilities

Currently supported: background plugins that integrate with the core experience. The platform roadmap includes dashboard widgets, custom layouts, and AI skill integrations.

Developers can build using standard web technologies (HTML, CSS, JavaScript/TypeScript) with framework flexibility—Vite, React, or vanilla JavaScript all work with the Even Hub SDK.

## Development Workflow

The process follows five steps: coding a standard web app, local preview via simulator, device testing through sideloading or portal upload, packaging the application, and submitting to Even Hub for distribution.

## Key Resources

Essential tools include the SDK, simulator, and CLI available through npm, plus design guidelines via Figma, community documentation on GitHub, and Discord support channels for developers.
