# Community Resources

## Overview

This documentation page highlights developer-contributed resources for the Even Realities platform, organized into three main sections.

## G2 Development Notes

A detailed reference guide maintained independently that covers:

- Architecture deep-dives
- Complete Unicode glyph tables
- SDK quirks and error codes
- Annotated examples from production applications

The resource includes reference implementations for six applications: chess, reddit, weather, tesla vehicle status, pong, and snake. Each demonstrates different patterns (modular architecture, API proxies, image-based rendering, canvas games, settings UI).

**Link:** [even-g2-notes on GitHub](https://github.com/nickustinov/even-g2-notes/blob/main/G2.md)

## Even Toolkit

A component library for building companion web interfaces alongside glasses applications. The toolkit provides:

**Components:** Button, Card, NavBar, ListItem, Toggle, Dialog, Toast, BottomSheet, Charts, Calendar, and additional UI elements

**Utilities:** Glasses bridge tools including `useGlasses` hook, `buildActionBar`, `mapGlassEvent`, canvas renderer, PNG utilities, and pagination helpers

**Design System:** CSS custom properties for colors, spacing, border radii, and fonts (FK Grotesk Neue), with light and dark theme support

**Typography:** Text size classes ranging from `.text-vlarge-title` (24px) to `.text-detail` (11px)

**Installation:**

```bash
npm install even-toolkit
```

**Link:** [even-toolkit on GitHub](https://github.com/fabioglimb/even-toolkit)
